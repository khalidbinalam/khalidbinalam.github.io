# Khalid Bin Alam - Portfolio Website

Professional portfolio website showcasing work as an Electronic Artist, Audio Engineer, and Curator.

## Features

- Single-page responsive design
- Multi-project identity showcase (Apurba, Zalvix, Luminous Lullabies)
- PlayLSD curation ecosystem
- Professional audio engineering portfolio
- Integrated social and streaming platform links

## Tech Stack

- Static HTML/CSS/JavaScript
- Vite for build optimization
- GitHub Pages deployment

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Deployment

This site is configured for GitHub Pages deployment. The site will be served from the root directory.

## Structure

- `index.html` - Main portfolio page
- `vite.config.js` - Build configuration
- `.env` - Environment variables (not committed)
